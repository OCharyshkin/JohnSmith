JohnSmith is a lightweight and simple JavaScript framework that works in conjunction with jQuery. JohnSmith allows you to use all the features of jQuery while following separation of concerns principle with MVVM pattern.

Getting started

To get started you need to reference jQuery and JohnSmith libs:

<script src="http://ajax.googleapis.com/ajax/libs/jquery/2.0.0/jquery.min.js"></script>
<script src="john-smith.debug.js"></script>

When DOM is ready you can construct views, configure bindings etc:

<script>
    $(function(){
        // ..JohnSmith code goes here...
    });
</script>
